CROPSAPP

Main menu where you can see your profile, your plots and manage them.

Inside 'Climate' you can choose among a several options to see a precise climate chart developed using NASA's database.

Here the user can administer his plots with the exact location and crop type. Doing this enables the app to create an optimized irrigation system depending on the type of crop and the characteristics of the plot.

Clicking in one of tour crops will give you access to detailed information: humidity, temperature,precipitations, UV index, wind and acidity.

If you click on irrigation you will be able to see the data from each plot and be able to start to irrigate if any of the plots needs it. the application will send an alert directly to the user's phone whenever is need.

You will have acces to a lot of crops data.
